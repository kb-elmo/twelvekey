# Twelvekey PCB

### Features

- In-switch LEDs
- Powered by an Atmega328p
- Basic ESD and overvoltage protection
- Centered USB type-C port for a detachable cable
- Fully programmable using [QMK](https://qmk.fm) and [Vial](https://get.vial.today)
